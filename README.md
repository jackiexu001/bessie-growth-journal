# Bessie 的成长日记 🌸

一个精美的成长记录网站，用于永久保存和展示宝宝成长过程中的照片和视频。

## ✨ 功能特点

- 📸 **照片和视频上传** - 支持多种格式的媒体文件上传
- 🎨 **精美展示** - 现代化的UI设计，响应式布局，支持移动端
- 📅 **时间线视图** - 按时间顺序展示所有回忆
- 🖼️ **相册画廊** - 网格布局展示所有照片和视频
- 💾 **本地存储** - 文件保存在本地，数据使用JSON存储
- 🚀 **易于扩展** - 基于Next.js 14，支持未来功能迭代

## 🛠️ 技术栈

- **框架**: Next.js 14 (App Router)
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **动画**: Framer Motion
- **图标**: Lucide React
- **日期处理**: date-fns

## 📦 安装和运行

### 1. 安装依赖

```bash
npm install
```

### 2. 运行开发服务器

```bash
npm run dev
```

打开 [http://localhost:3000](http://localhost:3000) 查看网站。

### 3. 构建生产版本

```bash
npm run build
npm start
```

## 📝 使用说明

### 上传照片/视频

1. 点击导航栏的"上传"按钮
2. 选择要上传的文件（支持图片和视频）
3. 填写标题、描述、日期和地点等信息
4. 点击"上传回忆"按钮

### 查看回忆

- **首页**: 查看时间线视图，按时间顺序展示所有回忆
- **相册**: 查看网格布局的照片和视频画廊
- 点击任意回忆可以查看大图或播放视频

## 🔒 隐私和安全

- 所有文件存储在本地，不会上传到第三方服务
- 建议定期备份 `public/uploads` 和 `data/memories.json` 目录
- 如需部署到公网，请添加身份验证功能

## 📦 部署建议

### Vercel部署

1. 将代码推送到GitHub
2. 在Vercel导入项目
3. 配置环境变量（如需要）
4. 注意：Vercel的免费计划不支持文件系统写入，需要集成云存储

### 自托管

1. 使用VPS或云服务器
2. 安装Node.js和npm
3. 克隆项目并安装依赖
4. 运行 `npm run build && npm start`
5. 配置Nginx反向代理
6. 设置SSL证书（Let's Encrypt）

## 💡 注意事项

1. **文件大小限制**: 当前设置为1024MB（1GB），可在 `app/api/upload/route.ts` 中修改
2. **存储空间**: 确保服务器有足够的存储空间
3. **备份**: 定期备份上传的文件和数据文件
4. **性能**: 如果文件数量很多，建议使用数据库和云存储
