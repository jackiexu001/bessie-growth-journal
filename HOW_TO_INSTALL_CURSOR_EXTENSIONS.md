# 如何在 Cursor 中安装扩展 📦

## 🎯 安装扩展的步骤

### 方法一：通过扩展市场安装（推荐）

#### 步骤 1：打开扩展市场

1. **打开 Cursor**
2. **点击左侧边栏的扩展图标**（四个方块的图标）
   - 或者按快捷键：`Command + Shift + X`（Mac）或 `Ctrl + Shift + X`（Windows）

#### 步骤 2：搜索扩展

1. **在搜索框中输入**：`Claude Code` 或 `claude`
2. **查看搜索结果**
3. **找到你想要的扩展**

#### 步骤 3：安装扩展

1. **点击扩展名称**
2. **点击 "Install"**（安装）按钮
3. **等待安装完成**

---

### 方法二：通过命令面板安装

#### 步骤 1：打开命令面板

1. **按快捷键**：`Command + Shift + P`（Mac）或 `Ctrl + Shift + P`（Windows）
2. **输入**：`Extensions: Install Extensions`
3. **按回车**

#### 步骤 2：搜索并安装

1. **在搜索框中输入**：`Claude Code`
2. **选择扩展**
3. **点击 "Install"**

---
