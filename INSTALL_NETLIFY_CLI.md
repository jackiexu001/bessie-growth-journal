# 安装 Netlify CLI 步骤 📦

## ⚠️ 当前状态

Netlify CLI 还没有安装，需要先安装。

---

## ✅ 安装步骤

### 步骤 1：使用 sudo 安装

在终端中运行：

```bash
sudo npm install -g netlify-cli
```

**重要提示：**
- 会提示输入你的 Mac 密码
- 输入密码时**不会显示任何字符**（这是正常的安全机制）
- 直接输入密码，然后按回车

### 步骤 2：等待安装完成

安装过程可能需要 1-2 分钟，请耐心等待。

### 步骤 3：确认安装

安装完成后，运行：

```bash
netlify --version
```

**如果安装成功，会显示版本号：**
```
netlify-cli/17.0.0
```

---

## ⚠️ 如果遇到问题

### 问题 1：提示密码错误

**解决：**
- 确认输入的是你的 Mac 登录密码
- 密码输入时不会显示字符，这是正常的
- 仔细输入，然后按回车

---

### 问题 2：安装很慢

**解决：**
- 这是正常的，安装可能需要 1-2 分钟
- 请耐心等待
- 不要中断安装过程

---

### 问题 3：安装失败

**解决：**
- 检查网络连接
- 尝试重新运行安装命令
- 如果还是失败，告诉我错误信息
