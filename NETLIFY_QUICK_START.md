# Netlify + Cloudinary 快速开始 🚀

## ✅ 好消息：代码不需要修改！

你的代码已经支持 Cloudinary，**不需要任何修改**！

只需要：
1. 配置环境变量
2. 部署到 Netlify

---

## 📋 部署步骤（简化版）

### 第一步：注册 Cloudinary（5分钟）

1. 访问 https://cloudinary.com
2. 注册账号
3. 获取 API 密钥：
   - Cloud name
   - API Key
   - API Secret

### 第二步：准备 GitHub（5分钟）

1. 创建 GitHub 仓库
2. 推送代码

### 第三步：部署到 Netlify（10分钟）

1. 访问 https://www.netlify.com
2. 使用 GitHub 登录
3. 导入项目
4. 配置环境变量
5. 点击部署

---

## 🔧 环境变量配置

在 Netlify 中添加以下环境变量：

```
STORAGE_TYPE=cloudinary
CLOUDINARY_CLOUD_NAME=你的cloud_name
CLOUDINARY_API_KEY=你的api_key
CLOUDINARY_API_SECRET=你的api_secret
```
