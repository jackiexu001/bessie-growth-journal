# 如何在 Cloudinary 中找到 API 密钥 🔑

## 📋 需要找到的信息

你需要找到以下三项：
1. **Cloud name**（云名称）
2. **API Key**（API 密钥）
3. **API Secret**（API 密钥）

---

## ✅ 详细步骤

### 步骤 1：登录 Cloudinary

1. 打开浏览器
2. 访问：**https://cloudinary.com**
3. 点击右上角 **"Log In"**（登录）
4. 输入你的邮箱和密码
5. 点击 **"Log In"**

---

### 步骤 2：进入 Dashboard（仪表板）

登录后，你会自动进入 **Dashboard**（仪表板）页面。

**如果没有自动进入：**
1. 点击左上角的 **"Dashboard"** 或 **"控制台"**
2. 进入 Dashboard 页面

---

### 步骤 3：找到 API 密钥信息

在 Dashboard 页面，你会看到以下信息：

#### 位置 1：页面顶部（最常见）

在 Dashboard 页面的**顶部**，你会看到一个信息卡片，显示：

```
Account Details
├── Cloud name: abc123
├── API Key: 123456789012345
└── API Secret: abcdefghijklmnopqrstuvwxyz
```

#### 位置 2：Account Details（账户详情）

1. 在 Dashboard 页面
2. 找到 **"Account Details"**（账户详情）部分
3. 点击展开或查看

#### 位置 3：Settings（设置）

1. 点击页面右上角的**设置图标**（齿轮图标）
2. 或者点击 **"Settings"**（设置）
3. 找到 **"Security"**（安全）或 **"Account"**（账户）
4. 查看 API 密钥信息

---

## 🔍 详细查找方法

### 方法一：Dashboard 页面顶部

1. **登录 Cloudinary**
2. **进入 Dashboard**
3. **查看页面顶部**

你会看到一个信息卡片，显示：
- **Cloud name**（云名称）
- **API Key**（API 密钥）
- **API Secret**（API 密钥）- 可能需要点击 "Show" 或 "显示" 才能看到

---

### 方法二：Settings → Security

1. **登录 Cloudinary**
2. **点击右上角的设置图标**（齿轮图标）
3. **选择 "Settings"**（设置）
4. **点击 "Security"**（安全）
5. **找到 "API Keys"**（API 密钥）部分

你会看到：
- **Cloud name**
- **API Key**
- **API Secret**（可能需要点击 "Show" 才能看到）

---

### 方法三：Account Settings

1. **登录 Cloudinary**
2. **点击右上角的头像或用户名**
3. **选择 "Account Settings"**（账户设置）
4. **找到 "API Keys"**（API 密钥）部分

---

## ⚠️ 重要提示

### API Secret 可能需要点击才能显示

- **API Secret** 通常是隐藏的（显示为 `******`）
- 需要点击 **"Show"**（显示）或 **"Reveal"**（显示）按钮
- 或者点击眼睛图标 👁️
- 才能看到完整的 API Secret

---

## 📋 找到后的操作

### 步骤 1：复制信息

1. **Cloud name**: 复制（例如：`abc123`）
2. **API Key**: 复制（例如：`123456789012345`）
3. **API Secret**: 点击 "Show" 后复制（例如：`abcdefghijklmnopqrstuvwxyz`）

### 步骤 2：保存信息

**重要：** 把这些信息保存到安全的地方（文本文件或密码管理器）

## ❓ 如果找不到

### 检查清单：

- [ ] 是否已登录 Cloudinary？
- [ ] 是否在 Dashboard 页面？
- [ ] 是否看到了 Account Details？
- [ ] 是否点击了 "Show" 查看 API Secret？

### 如果还是找不到：

1. **尝试搜索**: 在页面中使用 `Command + F`（Mac）搜索 "API Key"
2. **查看设置**: 点击右上角设置图标 → Settings → Security
3. **查看帮助**: 访问 Cloudinary 帮助文档
